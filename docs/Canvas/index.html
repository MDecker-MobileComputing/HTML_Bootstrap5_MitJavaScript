Canvas-Element und Bootstrap 5

Mit dem Canvas-Element von HTML5 können beliebige geometrische Formen gezeichnet werden – probieren Sie es aus!

  • Diagonalen
  • Dreieck
  • Rechteck
  • Kreis
  • Ellipse
  • Bézierkurve
Diese Option gilt natürlich nur für Formen, die eine geschlossene Fläche bilden, z.B. Kreis oder Ellipse. Die Option wird nur bei Neu-Auswahl von Formen berücksichtigt.
Ihr Browser unterstützt das Canvas-Element (Zeichenfläche) von HTML5 nicht.
